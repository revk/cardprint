JSON messages for xid server...

To server

"move":position		// Move to position

From server

"id":id			// Printer name
"type:type		// Printer type
"position":position	// At position
"error":{"description:...,"code"...}
"print":[side1,side2] or "print":side
side contains "C","M","Y","K","U" that are base64 coded data, and "uvsingle" boolean
